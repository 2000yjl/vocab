WordForge deploy notes

Upload all files in this folder to the ROOT of the GitHub repository.
Netlify will redeploy automatically after GitHub is updated.

This version has:
- No account system.
- No password system.
- No points or locked dictionaries.
- Local progress saved in the browser with localStorage.
- English, Korean, Japanese, German, Spanish and French learning modes.
- English US/UK pronunciation, IPA, word roots, related words, review queue, quiz, mistakes and pet companion.
